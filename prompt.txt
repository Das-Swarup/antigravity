# Prompt 1 – Neon Snake Game

```
Build a NodeJS-based (Node version compatible from v6 to latest) neon-themed Snake web app.

Create the project inside a new folder named `neon-snake`.

The UI must have:
- Dark background
- Cyan glowing snake
- Magenta glowing food
- Subtle grid background
- Smooth glowing animation using CSS shadows and transitions

Program the game logic in NodeJS using Express to serve static files.

The frontend should:
- Handle snake movement with arrow keys
- Detect food collision
- Grow the snake
- Detect wall or self collision
- Show a dramatic neon "Game Over" message at the center

Keep all JavaScript compatible with older Node versions (avoid ES6+ syntax).

Once the application is ready, tell me how to:
1. Install dependencies
2. Run it locally
3. Access it in the browser
```
